# homebrew-rimsort

Homebrew tap for [RimSort](https://github.com/RimSort/RimSort), an open source mod manager for RimWorld.

## Install

```bash
brew install TWDickson/rimsort/rimsort
```

## Update

```bash
brew upgrade rimsort
```

## Uninstall

```bash
brew uninstall rimsort
```

The cask is automatically updated every 6 hours via GitHub Actions when a new RimSort release is published.
